Détails sur la structure de données installée par le CD

On note DEVINT le répertoire spécifié au CD lors de l’installation pour y placer les projets DeViNT, par exemple DEVINT = C:\MesDocuments, et on suppose qu’on installe le seul jeu Motus par exemple. Dans ce cas, le CD crée le dossier C:\MesDocuments\Devint\2011 dans lequel il va copier le sous-dossier Motus accompagné de plusieurs sous-dossiers nécessaires à l’exécution de Motus.

  • Il y aura le dossier ‘aide’ qui contient une page aide.html avec un lien vers la page C:\MesDocuments\Devint\2011\Motus\doc\manuel.html qui contient l’aide de Motus.
  • Le dossier ‘jre’ contientles outils pour exécuter les projets écrits en langage java, c’est le Java Runtime Environment version 6 de Sun Microsystems, ou JRE, version 32 bits.
  • Le dossier lib contient les librairies dynamiques (extension dll) nécessaires pour l’exécution de certains projets ou périphériques sous Windows
  • Le dossier Listor contient un menu accessible pour lancer tous les projets installés
  • Le dossier Utilitaires est vide, il est créé lors de la première exécution pour éviter de répéter certaines opérations
  • Le dossier VocalyzeSIVOX contient la synthèse vocale accessible à tous les projets.
  • Le dossier Motus, ainsi que tous les projets installés, un dossier par projet
  • Le fichier exécutable Uninstall.exe capable de désinstaller tous les projets Devint en une fois ; en pratique, il efface tous les dossiers créés ainsi que les deux icônes placées sur le bureau par l’installation.
  • L’ordinateur de l’utilisateur du CD se retrouve alors dans l’état qu’il avait avant l’installation, excepté pour les utilitaires installés pour certains jeux (comme le tapis de danse, ou la webcam).
