Utilisation des projets sous Windows

A priori, les projets DeViNT sont conçus pour être utilisés sous Windows.

L’installation du CD a été testée extensivement sous Windows XP, où elle ne pose aucun problème, et récemment sous Windows Seven 32 bits et 64 bits.

Il est essentiel pour Windows Seven d’installer les projets dans une zone du disque dur où l’utilisateur a le droit d’écriture, ceci pour la synthèse vocale en particulier.

Installer dans le dossier ‘Program Files’ fonctionnait avec Windows XP parce qu’on pouvait écrire dans Program Files. Ce n'est pas le cas pour le dossier Program Files (x86) où les projets sont installés par défaut par Windows Seven.

Egalement, on a dû forcer l’installation du jre dans le dossier des projets DeViNT sur Windows Seven, pour ne pas avoir à modifier la variable PATH sous Windows Seven,

Enfin, on a dû placer les librairies dynamiques (dll 32 bits) dans le dossier des projets DeViNT, car elles n’étaient pas accessibles dans C:\Windows\System32.

Les dossier ‘jre’ contient le JRE 6 version 32 bits, et le dossier ‘lib’ contient les librairies dll 32 bits utilisés par certains projets.

Utilisation des projets sous Linux

Certains projets du CD ont été testés sous Linux 32 bits, en traduisant le script bin\execution.bat en un script shell bin/execution.sh et le script src/compilation.bat en src/compilation.sh. Il faut respecter la structuration des projets par rapport à la synthèse vocale. Des problèmes inexpliqués pour l'instant subsistent pour la synthèse vocale sous Linux 64 bits .

Utilisation sur MacOS

On nous a signalé que certains projets ont été testés sous MacOS. Nous sommes très demandeurs de tous les retours sur les possibilités et les limites des projets DeViNT.
